Teams API

Id: https://apigee.com/apis/3

Version: initial

The teams API allows for the creation and management of teams. Teams are simple lists of Users. teams are conceptually very similar to the SCIM groups provided by SSO2, except that they have a different access control model. The access control model for Teams is described in this document. This access control model allows Teams to be created without special permissions. The API for Permissions is here.
